Please install the font file 'aw_standard'. You will not be able to see any of the in-game text without so.

Should any errors pop up, they will be thrown into the 'Error.txt' file.
Please report any errors that do pop up to the forum: 

http://forum.chaos-project.com/index.php/topic,12581.0.html

To start the game, open 'Game.exe'. It should be an orange-sun-shaped icon.

Read the PDF file for detailed instructions on how to operate the game.



Thank you and enjoy!
